# 安装条件
## 操作系统
    Linux 或 Mac 或 window 系统
## python版本
    python3.7    
## 创建虚拟环境运行
    pip3 install pipenv
    pipenv --python 3.7 --创建虚拟环境
    pipenv shell -- 进入虚拟环境
    pipenv install -- 安装所有依赖
    pipenv graph  -- 查看安装了哪些依赖包 
## 非虚拟环境运行
    pip3 install -r requirements.txt -- 安装依赖
    
## 执行
    python3 maintest.py
